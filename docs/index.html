Classes

AppArray

Provides array functionality with removed restrictions on array size and element count.

AppBoolean

Implements Boolean variables and logic with unlimited instances for complex conditional operations.

AppCanvas

Represents a canvas that can be drawn on for the BOOSE interpreter, supporting shapes, lines, and text.

AppCommandFactory

Creates enhanced command instances with removed restrictions for the BOOSE interpreter.

AppCompoundCommand

Handles grouping of multiple commands with unlimited nesting capabilities.

AppIf

Provides conditional execution with unlimited if-statements for complex logic flows.

AppInt

Manages integer variables with expression evaluation and increased instance limits.

AppMethod

Enables custom method definition and calling with unlimited method declarations.

AppParser

Parses program text into executable commands without line count limitations.

AppPeek

Reads values from variables at runtime for debugging and monitoring purposes.

AppPoke

Modifies variable values at runtime for testing and debugging operations.

AppReal

Handles floating-point numeric variables with validation and expression parsing.

AppStoredProgram

Manages the program's commands and execution flow without instruction count limits.

AppWhile

Implements while-loop functionality with unlimited iterations for repetitive tasks.

AppWrite

Renders text on the canvas with custom positioning and styling capabilities.

BooseForm

Main application window that provides code editing, execution, and canvas display functionality.

BooseReset

Resets internal counters and restrictions to enable unlimited program execution.
